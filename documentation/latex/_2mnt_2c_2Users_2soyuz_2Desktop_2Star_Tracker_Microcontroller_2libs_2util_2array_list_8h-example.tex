A resizable array which can be declared on the stack. A stack array which has list like properties. This has a fixed size. In default conditions, this is treated to copy the value when inserting. This stops the requirement of new in a local scope.


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & The datatype to store. \\
\hline
{\em N} & The maximum size.\\
\hline
\end{DoxyTemplParams}
\begin{DoxyVerb}    ArrayList<int, 10> array(5);
    array.PushBack(123);
    cout << array.MaxSize() << endl;    // 10
    cout << array.Size() << endl;       // 6;
    cout << array.Get(5) << endl;       // 123;
    cout << array.PopBack() << endl;    // 123;
    array.ReduceSize(3);
    cout << array.Size() << endl;       // 3;
\end{DoxyVerb}
