libs/get_image

Obtaining the image, this currently only supports nix. It has been tested with the windows .bmp. Uses the library easybmp

How To Use

In the BUILD file, input:

deps = ["//libs/get_image:get_image"]

At the top of the desired file, input:

#include "libs/get_image/get_image.h"

Build

Input the following into your command line:

bazel build //libs/get_image/...

Test

Input the following into your command line:

bazel test //get_image/get_image/... --test_output=all
